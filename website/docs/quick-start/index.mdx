The ProAbono API Live lets you manage subscriptions programmatically — create customers, start or update subscriptions, check feature access, and calculate billing costs before committing to a change.

## Prerequisites

- A ProAbono account with an active Business
- Your **Agent Key** and **API Key** — find them in the backoffice under **Integration**
- Your **Business ID** (`IdBusiness`) — also in the Integration section

## Step 1 — Identify your base URL

Every Business has its own base URL:

```
https://api-<IdBusiness>.proabono.com/
```

See [Authentication](../authentication/) for authentication details.

## Step 2 — Create a customer

<OperationSnippets
  method="POST"
  path="/v1/Customer"
  baseUrl="https://api-<IdBusiness>.proabono.com"
  body={{ ReferenceCustomer: "user-1", Email: "alice@example.com", Name: "Alice" }}
/>

## Step 3 — Inspect the response

A successful request returns `201 Created` with the new customer object:

```json
{
  "Id": 42,
  "ReferenceCustomer": "user-1",
  "Email": "alice@example.com",
  "Name": "Alice",
  "Status": "Enabled",
  "Links": [{ "rel": "self", "href": "/v1/Customer/42" }]
}
```

See [Customers](../api-reference/customers) for the full object schema.
