ProAbono is a subscription-billing platform and API Live is designed to craft your end-users experience in a few calls.

The ProAbono API Live lets you manage subscriptions programmatically — create customers, start or update subscriptions, check feature access, and calculate billing costs before committing to a change.

:::warning
This API is designed to be used from a server to a server, and **NOT** from a client to server in JavaScript. This is very important for security purposes.
:::
